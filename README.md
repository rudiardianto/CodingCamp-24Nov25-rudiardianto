# To-Do List Web Application

Aplikasi ini dibuat untuk Mini Coding Project CodingCamp.

## Teknologi
- HTML
- Tailwind CSS
- JavaScript

## Fitur
- Menambahkan tugas baru
- Tandai tugas selesai / batal
- Hapus tugas
- Filter tugas: Semua / Selesai / Belum Selesai
- Data tersimpan di local storage

## Cara Menjalankan
1. Buka file `index.html` di browser (Google Chrome disarankan).
2. Tambahkan tugas baru, tandai selesai, hapus tugas, dan filter tugas sesuai kebutuhan
